Este repositório contém meus exercícios práticos em **Python**, desenvolvidos ao longo do meu processo de aprendizagem da linguagem.

O objetivo principal é fixar **conceitos fundamentais**, praticar lógica de programação e evoluir gradualmente para estruturas e ideias mais avançadas.
 ## Conteúdos Abordados 
 Os exercícios incluem, entre outros temas:
 * Entrada e saída de dados

* Variáveis e tipos primitivos

* Operadores

* Estruturas condicionais

* Estruturas de repetição

* Tuplas, listas e dicionários

* Funções e modularização

* Tratamento de erros e exceções (__*`try`, `except`, `else`, `finally`*__)

* Exercícios e desafios propostos ao longo do curso

* Exercícios de lógica e resolução de problemas
## Objetivo do Repositório 
 * Praticar Python de forma contínua

* Consolidar a base de lógica de programação

* Acompanhar minha evolução na linguagem

* Servir como material de consulta no futuro
